How to run (Windows)
1) Put these files into C:\icip_osr\code\
2) Ensure your conda env has open_clip_torch + torch + sklearn installed.
3) Double-click RUN_PAPER.bat (edit DATA_ROOT inside if needed).
Outputs:
 - results\<timestamp>\results.csv
 - results\<timestamp>\main_table_mean_std.csv
 - results\<timestamp>\macro_avg.csv
 - results\<timestamp>\delta_vs_baseline.csv
 - results\<timestamp>\paired_delta_summary.csv
 - results\<timestamp>\paper_tables.md
